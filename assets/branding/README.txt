# Promethium Assets - Branding

This directory contains branding assets for the Promethium framework.

## Contents

- `promethium-logo.png` - Primary logo (320x320, dark waveform on transparent)
- `promethium-logo-light.png` - Light variant for dark backgrounds
- `promethium-icon.png` - Square icon (64x64) for favicons

## Usage Guidelines

1. Maintain aspect ratio when scaling
2. Use appropriate variant for background color
3. Minimum clear space: 10% of logo width on all sides

## Colors

- Primary Dark: #1E1E2E
- Primary Blue: #3B82F6
- Accent: #10B981
